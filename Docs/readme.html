Installation Launcher

Open the project in Visual Studio 2013

Open WowSuite.Launcher. Double-clicking, open the "Properties" and go to the tab "Settings"

Edit the settings for your server

You can see the variables such as: Links to your project, Name your server, Links to the server part, Server and Mysql IP:PORT, Language of your game client,Realmlist settings and etc.

Attention!!! If you change a string "SKEY_API" - don't forget change this string in file "launcher_api.php" at 3 line

Open WowSuite.Constants. Double-clicking, open the "Wow.cs"

And change the line LOCALE_FOLDER_NAME in the language of your game client


Installation Server Side

Copy folder "launcher" from "Server Side" in the root of your Web site

Create 2 database "icontemplate" and "news"

Execute 3 sql files into you DB (hot_news.sql and news_launcher.sql => in "news" DB) (icontemplate.sql => in "icontemplate" DB)

Configure file config.php in "launcher/api/' folder

Open file "lang" in "website/launcher/files/" and change the language of your game client

INSERT INTO `news_launcher` VALUES ('1', 'HEAD NEWS', 'BODY NEWS', 'http://127.0.0.1/');

Where values "ID news", "Title news", "Text news", "Link to news on website"

Updating Game Client

In order to properly create a file of information for game client updates, needed:

  • Copy you patches (*.mpq) in "website/launcher/files/"
  • Run program "CreatePack". The program will show you the current version of the update as well as ask you to enter the version number of the update. Attention! You must enter the number of the next update.

Patch name should be:

If you want your patch was loaded in path of "Data", the name of the patch should be (For example) - Patch-9.mpq

If you want your patch was loaded in path of "enGB(or you lang)", the name of the patch should be (For example) - Patch-enGB-22.mpq


Game files

When you have compiled the launcher and set up Server Part, you should prepare a launcher for your players

After compiling launcher open - wowsuite-launcher/launcher/bin/Debug(or Release) and find 3 files

  • Launcher.exe
  • WowSuite.Constants.dll
  • WowSuite.Utils.dll

These three files you need to send for your players

Players must place the files within a directory to the root of the game
